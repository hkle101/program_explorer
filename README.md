# Program Explorer

The program explorer site is a website designed to help explore six of Purdue’s Applied and Creative Computing majors.
The page has six program cards, and they have course and career information displayed by bootstrap modals.
There is a quiz that recommends a program based on user input. The site uses css that is compiled from an scss file to add purdue colors as variables.
They are mainly used to style colors of elements across the page. The page is responsive by utilizing the bootstraps grid system to customize how the site will look on different screen sizes.
The hamburger menu is implemented using bootstrap’s responsive navbar to collapse when the screen width gets too small. Other bootstrap features used were cards, buttons, modals, and list groups classes to compact information in the page.
Accessibility is implemented through the use of aria attributes and appropriate semantic tags throughout the page to work with screen readers. Lastly the favicon used is a computer brain image to match the theme of the website.
